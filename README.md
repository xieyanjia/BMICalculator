# BMI計算機
這是一個使用 C# Windows Forms 開發的 BMI 計算工具，  
使用者可以輸入身高與體重，快速計算 BMI 並查看健康狀態，同時保留歷史紀錄。

## 功能介紹

- 輸入身高（公分）與體重（公斤）
- 點擊「計算」按鈕
- 即可顯示 BMI 數值與對應的健康分類
- 最後可以記錄每次的身高體重BMI數值與對應的健康分類
